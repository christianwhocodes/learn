## ğŸ‹ï¸ Exercise Structure

Each module with exercises includes three types of files:

- **`*-start.[ext]`**: Starting template with task descriptions and hints
- **`*-work.[ext]`**: Your workspace to complete the exercises
- **`*-end.[ext]`**: Complete solution for reference

### How to Use Exercises

1. Open the `*-start.[ext]` file to see the tasks
2. Work in the `*-work.[ext]` file (this is your practice space)
3. Check your work against `*-end.[ext]` when done

**Note**: The `*-work.[ext]` files are gitignored by default, so your practice work stays local.
